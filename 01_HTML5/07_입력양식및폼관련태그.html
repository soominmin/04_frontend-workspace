form 태그 내에 submit버튼 클릭시 해당 form안에 작성된
사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할 수행 form action= "" method=""

*action 속성: 해당 form 에 사용자가 입력한 값들을 전달하면서, 요청할 서버의 경로 제시
*mathod 속성: 요청 전송 방식을 지정하는 속성(기본:get, post)

>> get 방식 : 요청시 사용자가 입력한 값들이 url에 노출 되는 방식(빠름)
>> post 방식: 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식(개인정보)
form 내에 summit 버튼 클릭시 action 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청된
(사용자가 입력한 값들은 항상 key= value 세트로 전달)
