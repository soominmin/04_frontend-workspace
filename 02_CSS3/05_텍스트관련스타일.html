텍스트 관련 스타일 (text~)

color: 텍스트의 색상을 지정

선택자 {
    color: 색상명 | 16진수 | RGB(x,x,x) | RGBA(X,X,X,투명도) 
    | HSL(X,X,X)_명도채도(잘안씀) | HSLA(X,X,X,투명도)
}

text-decoration: 텍스트에 줄을 긋거나 없앨 때 사용

선택자 {
    text-decoration: none | underline | overline | line-through
}
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간 줄 긋기
  • 텍스트 영역 위 줄 긋기
네이버로

text-transform: 영문 텍스트 대소문자 변환시 사용

선택자 {
    text-transform: uppercase | lowercase | capitalize
}
  1. uppercase: 모든 영문자 다 대문자로
  2. LowerCase: 모든 영문자 다 소문자로
  3. capitalize: 영무자의 first text만 대문자로

text-shadow: 텍스트에 그림자 효과를 줄 때 사용

선택자 {
    text-shadow: x(가로거리) y(세로거리) [번짐정도] [색상]
}
